# proyectofrio PASOS PARA EJECUTARLO
1. DESCARGAR CARPETA PROYECTOFRIO Y INSTALAR VISUAL STUDIO CODE Y ABRIR CARPETA DESDE DENTRO
   
📂 2. Entrar en la carpeta del proyecto
cd proyectofrio

📦 3. Instalar dependencias
npm install

▶️ 4. Ejecutar el proyecto
npm run dev

🌐 5. Abrir en el navegador
(https://localhost:3000/) INFO::TIENE HTTP(S) PORQUE LLEVA UN CERTIFICADO PARA HACERLE CREER AL PROGRAMA DEL MODELO 3D QUE ES SEGURA PARA QUE ME DEJE COLOCAR LAS ETIQUETAS

🧠 Requisitos
Node.js instalado 👉 https://nodejs.org
npm (incluido con Node.js)
💡 Notas
Proyecto realizado con Vue + Vite
El puerto puede cambiar, se mostrará en la terminal al ejecutar
